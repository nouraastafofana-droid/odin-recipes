# odin-recipes
My First HTML project for the Odin Project

## Project Description

This project is part of The Odin Project curriculum and focuses on building my first basic website using HTML.

The goal is to create a simple recipe website that consists of a main index page linking to several individual recipe pages. Each recipe page contains structured content such as headings, paragraphs, images, and lists.

## Skills Demonstrated

By completing this project, I have demonstrated the following skills:

- Writing clean and structured HTML
- Using semantic HTML elements (headings, paragraphs, lists)
- Creating links between pages (relative links)
- Organizing files and folders properly
- Embedding images into a webpage
- Understanding basic HTML document structure (boilerplate)

## Reflection

This project helped me understand how to structure a website using only HTML. I learned how to link multiple pages together and how important file organization is when working with relative paths.

It also reinforced the importance of semantic HTML for readability and maintainability. This project serves as a strong foundation before moving on to CSS and more advanced web development concepts.
